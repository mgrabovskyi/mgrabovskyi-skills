# Productivity

Cross-cutting workflow tools — daily rituals, scheduled rollups, communication automation. Useful regardless of role.

- **[morning-briefing](./morning-briefing/SKILL.md)** — Configure a recurring daily rollup of Slack/Teams + email activity and emit a self-contained scheduled-task prompt that posts a four-section summary (asks, mentions, updates, blockers) each morning.
- **[session-handoff](./session-handoff/SKILL.md)** — Compact the current conversation into a structured leader-shaped handoff (decisions made, decisions open, asks, stakes, next-session prompt) and write it to a destination you pick: temp file, project state file, Notion page, or Slack canvas. Use when a multi-session leader workflow needs to be paused and resumed in a fresh chat.
