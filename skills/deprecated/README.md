# Deprecated

Skills that have been retired. Kept here so:

- Old install commands and links don't 404.
- The intent and prior shape of the skill are preserved for reference.
- A new contributor can see what's been tried and why it was retired.

Skills in this bucket **must not** appear in the top-level `README.md`. Each deprecated skill's `SKILL.md` should open with a short note explaining why it was deprecated and what (if anything) supersedes it.
